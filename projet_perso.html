Todo List - VueJS
Testez moi !
Contexte

TodoList est ma première mini application que j'ai réalisée en VueJs. Vue.js, est un framework JavaScript open-source utilisé pour construire des interfaces utilisateur et des applications web monopages. Cette Todo List est une application assez courte qui m'a permise de manipuler des listes en réalisant de simples actions pour commencer.

Mon portfolio
Contexte

J'ai réalisé moi-même ce portfolio dans le cadre de l'examen final du BTS SIO. Le but est d'y recenser tous mes projets personnels et scolaires ainsi que mes compétences et mes expériences professionnelles. Il a été développé en HTML, CCS et Javascript.

Fitactiv
Contexte

Fitactiv est un site web vitrine que j'ai réalisé récemment pour un client fictif. Le but était de me familiariser avec le CMS Wordpress. Le tutoriel sur lequel je me suis appuyé est celui d'Openclassrooms. Le projet à durer une petite semaine pendant laquelle j'ai appris à manipuler les extensions essentielles ainsi que la configuration d’un site WordPress.

How to computer - Symfony
Contexte

How To Computer est un site web réalisé en Symfony 4. Après avoir travaillé le PHP, j'ai découvert l'utilisation du Framework Symfony. Ce site n'avait pas de contexte particulier. L'objectif était de me familiariser avec une architecture MVC et d'en comprendre les principes. Bien sûr, je me suis aidée de tutoriels YouTube créer par Lior Chamla. Entre autres, j'ai appris à utiliser l'ORM Doctrine, le système de migration, les fixtures... J'ai également implementé des formulaires et enfin la découverte du composant Security de Symfony.

L.A Event - PHP
Contexte

L.A event anciennement Artists fait partie de mes tout premiers projets il y a maintenant 2 ans. C'est un mini blog fictif censé regrouper diverses actualités du monde artistique américain. L'objectif était d'acquérir les fondamentaux de PHP : Réalisation de formulaire d'inscription et de connexion et publication d'article.
