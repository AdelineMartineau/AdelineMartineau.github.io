How to computer - Symfony
Contexte

How To Computer est un site web réalisé en Symfony 4. Après avoir travaillé le PHP, j'ai découvert l'utilisation du Framework Symfony. Ce site n'avait pas de contexte particulier. L'objectif était de me familiariser avec une architecture MVC et d'en comprendre les principes. Bien sûr, je me suis aidé de tutoriels YouTube créer par Lior Chamla. Entre autres, j'ai appris à utiliser l'ORM Doctrine, le système de migration, les fixtures... J'ai également implementer des formulaires et enfin la découverte du composant Security de Symfony.
